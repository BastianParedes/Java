El programa fue desarrollado en Java con el IDE Visual Studio Code con la extensión "Extension Pack for Java" instalada.
Se tienen tres clases "Cliente", "Vehiculo y "Arriendo".
Cada clase tiene su propia base de datos guardadas en "clientes.txt", "vehiculos.txt" y "arriendos.txt"
Se evitó usar tildes y la letra ñ a menos que dischos elementos estuvieran en String.
El único IDE en el que funcionó esta aplicación fue en Visual Studio Code, ya que en otras como Eclipse no se reconocieron las bases de datos.
Al iniciar el script "Main.java" se desplegarán las opciones que se solicitaron en esta evaluación.

Al generar un arriendo, se debe tener en cuenta que el rut del cliente y la patente del vehículo deben estar ingresados en el sistema de manera previa.
